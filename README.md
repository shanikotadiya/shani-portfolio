# Shani Kotadiya Portfolio 🚀

A modern, responsive, and performance-focused developer portfolio built with **Next.js**, **React.js**, and **Tailwind CSS**.  
Designed to showcase projects, skills, experience, and provide a professional contact experience.

---

## 🌐 Live Demo

[Visit Portfolio](#)

# https://shani-portfolio-beta.vercel.app/

## ✨ Features

- ⚡ Built with Next.js App Router
- 🎨 Modern responsive UI
- 🌙 Clean dark theme
- 📱 Mobile-first design
- 🚀 Fast loading & optimized performance
- 🔍 SEO Friendly metadata setup
- 📂 Projects showcase section
- 💼 Skills & Experience section
- 📬 Working contact form with email integration
- ❌ Custom beautiful 404 page
- 🧠 Clean reusable component structure
